# 週次サマリールール（weekly-summary）

1週間分の情報を集約して `weekly/YYYY-Www.md` に出力。

## 入力ソース

- `daily/` — 該当期間のアジェンダ・メモ
- `inbox/meetings/` — 該当期間の議事録
- `inbox/tasks/` — 該当期間のタスクファイル
- Google Calendar — 該当週の予定一覧

## 出力フォーマット

```markdown
# 週次サマリー: YYYY年 第WW週（M/D〜M/D）

## 今週の数字
- MTG参加数: XX件
- 作業ブロック確保: XX時間
- タスク完了: XX件 / 新規発生: XX件

## 主な意思決定
1. [会議名 M/D] ○○を決定
2. ...

## MTG ダイジェスト

### 定例系
- プロモ×開発（月・木）: ...
- 開発定例（火・木）: ...

### 1on1・よもやま
- よもやま（火・木）: ...

### スポット
- [会議名]（曜日）: ...

## 未完了タスク
- [ ] ...

## 来週に向けて
- ○○の期限が来週
- △△のフォローアップが必要
```

## ルール

- `daily/` ファイルがない日は Google Calendar の予定だけで構成
- 議事録がない MTG は予定名のみ記載
- 来週の Google Calendar 予定も取得して「来週に向けて」を生成
- `knowledge/decisions.md` に今週の意思決定を追記する
