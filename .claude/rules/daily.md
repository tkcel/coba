# 日次アジェンダルール（daily-agenda）

Google Calendar から今日の予定を取得し、`daily/YYYY-MM-DD.md` を生成。

## 出力フォーマット

```markdown
# YYYY-MM-DD（曜日）

## スケジュール

| 時間 | 予定 | 種別 | 人数 |
|------|------|------|------|
| 09:00-09:30 | 朝礼 | 全体朝礼 | 20名 |
| 10:00-11:00 | 開発定例 | 定例MTG | 7名 |

## MTG メモ欄

### 09:00 朝礼
-

### 10:00 開発定例
- 📎 https://...（カレンダーのdescriptionから）
-

## 作業ブロック
- 13:00-17:00 集中作業
  - [ ]

## 持ち越し TODO
- [ ] ...（前日・inbox/tasks/ から）
```

## MTG 分類ルール

| パターン | 種別 |
|----------|------|
| 「朝礼」 | 全体朝礼 |
| 「定例」「【定例】」 | 定例MTG |
| 「よもやま」「1on1」 | 1on1 |
| 「開発定例」「開発✖」 | 開発系MTG |
| 「外部」「商談」 | 外部MTG |
| outOfOffice | 作業ブロック |
| 参加者1名 | 個人ワーク |
| それ以外 | その他MTG |

## ルール

- eventType が `outOfOffice` → 「作業ブロック」セクションに分離
- description にURLがあれば 📎 付きで記載
- 既に同日のファイルがある場合は上書きせず確認する
- `inbox/tasks/` に未完了タスクがあれば「持ち越し TODO」に記載
