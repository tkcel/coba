# 日次アジェンダルール（generate-daily）

Google Calendar から今日の予定を取得し、`daily/YYYY-MM-DD.md` を生成。

## 出力フォーマット

```markdown
# YYYY-MM-DD（曜日）

## TODO

ステータス別にグループ化して表示。`inbox/tasks/current.md`（マスター）から自動生成。

### 🔲 未着手
| タスク | 期限 | 備考 |
|--------|------|------|
| ○○の対応 | 要確認 | - |

### 📝 進行中
| タスク | 期限 | 備考 |
|--------|------|------|
| ○○の対応 | MM/DD | メモなど |

### ✅ 完了
なし

### ⌛️ 保留
なし

## スケジュール

\`\`\`mermaid
gantt
    title YYYY-MM-DD（曜日）
    dateFormat HH:mm
    axisFormat %H:%M

    朝礼           :09:00, 09:30
    チーム定例       :10:00, 11:00
\`\`\`

## MTG メモ欄

### 09:00 朝礼
-

### 10:00 チーム定例
- 📎 https://...（カレンダーのdescriptionから）
-

## 作業ブロック
- 13:00-17:00 集中作業
  - [ ]
```

## MTG 分類ルール

### デフォルトルール

| パターン | 種別 |
|----------|------|
| outOfOffice | 作業ブロック |
| 参加者1名 | 個人ワーク |
| それ以外 | その他MTG |

### カスタムルール

<!-- /onboarding または /optimize-rules で追加 -->
| パターン | 種別 |
|----------|------|
| 「朝礼」「全社朝礼」 | 朝礼 |
| 「開発定例」 | 開発定例 |
| 「定例」「【定例】」「[定例]」 | 定例MTG |
| 「よもやま」「1on1」 | 1on1 |
| 「絶対作業時間」「ブロック」 | ブロック（除外） |

## Jira セクション

`## 作業ブロック` の後に `## Jira` セクションを配置。

```markdown
## Jira

### オープンチケット
| キー | タイトル | ステータス | 優先度 | 期限 |
|------|---------|----------|--------|------|
| PROJ-123 | ○○の実装 | In Progress | High | 03/15 |

### 今日更新
| キー | タイトル | 変更内容 |
|------|---------|---------|
| PROJ-789 | ××の修正 | Done に変更 |
```

- `scripts/jira.sh search` で自分のオープンチケットと今日更新分を取得
- Jira 連携が未設定・取得失敗時は「Jira: 取得できませんでした」と記載してスキップ
- `.claude/rules/jira.md` のフォーマットに従う

## ルール

- eventType が `outOfOffice` → 「作業ブロック」セクションに分離
- description にURLがあれば 📎 付きで記載
- 既に同日のファイルがある場合は上書きせず確認する
- `inbox/tasks/current.md`（マスター）から未完了タスクを集約し、先頭の「TODO」セクションにダッシュボードとして表示
- **自分（`knowledge/me.md` のオーナー）が担当のタスクのみ表示**。他の人のタスクは含めない
- 完了済み（`- [x]`）のタスクは表示しない
- デイリーの TODO は読み取り専用。タスクの完了・追加は `inbox/tasks/current.md` で行う
