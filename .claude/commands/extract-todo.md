# /extract-todo

指定されたファイル、または inbox/meetings/ 内の議事録からアクションアイテムを抽出する。

## 引数

- `$ARGUMENTS` - 抽出対象のファイルパス（省略時は inbox/meetings/ 内の最新議事録を対象）

## 抽出対象

- 明示的なTODO:「○○する」「○○を確認」「○○を依頼」
- 決定事項から派生するアクション:「○○に決定 → 担当者が対応」
- 宿題・持ち帰り:「次回までに」「確認して共有」「要フォロー」

## 出力フォーマット

```markdown
# タスク一覧（YYYY-MM-DD 抽出）

## [出典ファイル名]
- [ ] タスク内容（担当: XX, 期限: MM/DD）
- [ ] タスク内容（担当: 要確認, 期限: 要確認）
```

## 出力先

`inbox/tasks/YYYY-MM-DD_tasks.md`

## ルール

- 担当者・期限が不明な場合は「要確認」と記載
- `knowledge/people.md` を参照して担当者名を統一
- 複数ファイルを一括処理する場合は、出典ごとにセクションを分ける
