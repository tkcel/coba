# /extract-todo

議事録やメモからアクションアイテムを抽出する。

## 対象

- `$ARGUMENTS` で指定されたファイル
- 指定なしの場合は `inbox/meetings/` 内の整形済み議事録

## 抽出対象

- 明示的なTODO:「○○する」「○○を確認」「○○を依頼」
- 決定事項から派生するアクション:「○○に決定 → 担当者が対応」
- 宿題・持ち帰り:「次回までに」「確認して共有」「要フォロー」

## 出力フォーマット

```markdown
# タスク一覧（YYYY-MM-DD 抽出）

## [出典ファイル名]
- [ ] タスク内容（担当: XX, 期限: MM/DD）
- [ ] タスク内容（担当: 要確認, 期限: 要確認）
```

## ルール

- 担当者・期限が不明な場合は「要確認」と記載
- `knowledge/people.md` を参照して担当者名を統一
- 複数ファイルを一括処理する場合は、出典ごとにセクションを分ける
- 出力先: `inbox/tasks/YYYY-MM-DD_tasks.md`
