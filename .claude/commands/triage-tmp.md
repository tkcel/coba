# /triage-tmp

tmp/ フォルダの中身を確認し、inbox/ の適切なサブフォルダに振り分ける。

## 手順

1. `tmp/` 内のファイルを一覧表示
2. 各ファイルの内容を確認し、種別を判定
3. ユーザーに振り分け先を確認
4. 確認後、`inbox/` 配下に移動

## 振り分けルール

| 判定 | 振り分け先 | 判定基準 |
|------|-----------|----------|
| 会議メモ | inbox/meetings/ | 「会議」「MTG」「定例」「議事録」、日時+参加者の記述 |
| タスク | inbox/tasks/ | 「TODO」「タスク」「やること」「期限」が主な内容 |
| メモ | inbox/notes/ | 上記に該当しないテキスト |
| 不明 | inbox/other/ | 判定できないもの |

## ファイル名ルール

- 移動先: `YYYY-MM-DD_元のファイル名`
- 元のファイル名に日付が既にあればプレフィックス不要
- 同名ファイルが存在する場合は末尾に `_2`, `_3` と連番

## 注意

- ファイルは**移動**する（コピーではない）
- **削除は絶対にしない**
- 完了後サマリーを報告
