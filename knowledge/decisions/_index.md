# 意思決定ログ

会議で決まった重要な意思決定を1件1ファイルで蓄積する。

## ファイル命名規則

```
YYYY-MM-DD_slug.md
```

例: `2026-03-10_release-date.md`

## ファイルフォーマット

```markdown
# [決定事項のタイトル]

- **日付**: YYYY-MM-DD
- **会議**: 会議名
- **関係者**: @名前, @名前

## 決定内容

何を決めたか

## 背景・理由

なぜその決定に至ったか（任意）

## 関連リンク

- [[関連する決定]]（任意）
```

## 使い方

- 週次サマリー生成時に自動作成
- 「decisions に○○追加して」で手動作成もOK
- 重要な意思決定のみ記録（日常的な決め事は不要）
- 後から「あの件どうなったっけ？」と検索できることが目的
