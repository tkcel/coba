# 社内用語辞書

議事録やメモに出てくる略称・社内用語を正式名称に変換するための辞書。

## 使い方
- 「glossary に CA を追加して」と言えば自動追記
- 議事録整形時に未知の略称があれば「これ追加する？」と提案

## 記入例

| 用語・略称 | 正式名称・意味 |
|-----------|---------------|
| CA | キャリアアドバイザー |
| PdM | プロダクトマネージャー |
| MTG | ミーティング |
| FYI | 参考情報（For Your Information） |
| ASAP | できるだけ早く |
| WIP | 作業中（Work In Progress） |
| TBD | 未定（To Be Determined） |
